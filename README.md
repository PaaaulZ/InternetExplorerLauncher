# InternetExplorerLauncher (We Want Internet Explorer)
Allows you to launch an url with Internet Explorer since launching iexplore.exe directly now opens Edge

Usage: ```wwie.exe [url]``` 

Example: ```wwie.exe https://www.google.com```
